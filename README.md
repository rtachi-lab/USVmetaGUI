# USV Meta GUI

PySide6-based desktop GUI for creating mouseTube-compatible USV metadata JSON files.

## Features

- Create one JSON per recording
- Reuse experiment templates
- Reuse user presets
- Preview JSON before saving
- Save mouseTube-compatible JSON locally

## Run

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
